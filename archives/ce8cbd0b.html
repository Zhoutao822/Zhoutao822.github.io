1. 正则化学习目标

假定数据集有nnn个样本，每个样本有mmm个特征，D={(xi,yi)}(∣D∣=n,xi∈Rm,yi∈R)\mathcal{D} = \{ (\boldsymbol{x}_i, y_i) \} (|\mathcal{D}| = n, \boldsymbol{x}_i \in \mathbb{R}^m, y_i \in \mathbb{R})D={(xi​,yi​)}(∣D∣=n,xi​∈Rm,yi​∈R)，并且该数据集属于回归任务，特征值和预测目标都是任意常数。假定用KKK个树构建模型，并且预测结果为所有树的预测结果的和

y^i=ϕ(xi)=∑k=1Kfk(xi),fk∈F\hat{y}_i = \phi(\boldsymbol{x}_i) = \sum^K_{k=1}f_k(\boldsymbol{x}_i), \quad f_k \in \mathcal{F} y^​i​=ϕ(xi​)=k=1∑K​fk​(xi​),fk​∈F

其中F={f(x)=wq(x)}(q:Rm→T,w∈RT)\mathcal{F} = \{ f(\boldsymbol{x}) = w_{q(\boldsymbol{x})} \}(q:\mathbb{R}^m \rightarrow T, w \in \mathbb{R}^T)F={f(x)=wq(x)​}(q:Rm→T,w∈RT)，TTT是当前树的叶结点数（我们将叶结点编号），qqq是从样本x\boldsymbol{x}x到叶结点index的映射，即样本x\boldsymbol{x}x落在树的index叶结点上，wq(x)w_{q(\boldsymbol{x})}wq(x)​表示该样本属于的叶结点的权值，我们可以将fff看作是一棵树的表示函数，其包括两个部分：树的结构函数qqq和树的叶结点权重www。

XGBoost是由多个决策树相加形成的，考虑增加正则项Ω\OmegaΩ，则整个模型的优化目标为

L(ϕ)=∑il(y^i,yi)+∑kΩ(fk)Ω(f)=γT+12λ∣∣w∣∣2\mathcal{L}(\phi) = \sum_i l(\hat{y}_i, y_i) + \sum_k \Omega(f_k) \\ \Omega(f) = \gamma T + \frac{1}{2}\lambda ||w||^2 L(ϕ)=i∑​l(y^​i​,yi​)+k∑​Ω(fk​)Ω(f)=γT+21​λ∣∣w∣∣2

前半部分是预测值与真实值的误差，后半部分是对树的结构的惩罚，Ω\OmegaΩ前半部分是对叶结点数的惩罚（避免过拟合严重），后半部分是对叶结点权重的惩罚（L2L_2L2​范数），也就是说XGBoost在回归树的基础上增加了对结点的权重惩罚（回归树结点取样本均值）。

2. 决策树梯度上升

我们知道神经网络的反向传播是应用于所有参数的，也就是说神经网络的参数更新是对所有的参数；XGBoost的梯度上升或下降是通过加法实现的，也就是说通过构建下一棵树来实现对当前树的梯度下降，我们在第ttt轮（这里ttt与上面的kkk几乎等价）的优化目标为

L(t)=∑i=1nl(yi,y^i(i−1)+ft(xi))+Ω(ft)\mathcal{L}^{(t)} = \sum^n_{i=1}l(y_i, \hat{y}_i^{(i-1)} + f_t(\boldsymbol{x}_i)) + \Omega(f_t) L(t)=i=1∑n​l(yi​,y^​i(i−1)​+ft​(xi​))+Ω(ft​)

我们仅考虑加入ftf_tft​以优化模型，这种做法是贪心的。通过二阶近似f(x+△x)=f(x)+f′(x)△x+12f′′(x)(△x)2f(x+\bigtriangleup x) = f(x) + f'(x)\bigtriangleup x + \frac{1}{2}f''(x) (\bigtriangleup x)^2f(x+△x)=f(x)+f′(x)△x+21​f′′(x)(△x)2，可以将上式近似表示为

L(t)≃∑i=1n[l(yi,y^i(t−1))+gift(xi)+12hift2(xi)]+Ω(ft)\mathcal{L}^{(t)} \simeq \sum^n_{i=1}[l(y_i, \hat{y}_i^{(t-1)})+ g_if_t(\boldsymbol{x}_i) + \frac{1}{2}h_if_t^2(\boldsymbol{x}_i)] + \Omega(f_t) L(t)≃i=1∑n​[l(yi​,y^​i(t−1)​)+gi​ft​(xi​)+21​hi​ft2​(xi​)]+Ω(ft​)

其中gi=∂y^i(t−1)l(yi,y^i(t−1))g_i = \partial_{\hat{y}_i^{(t-1)}}l(y_i, \hat{y}_i^{(t-1)})gi​=∂y^​i(t−1)​​l(yi​,y^​i(t−1)​)，hi=∂y^i(t−1)2l(yi,y^i(t−1))h_i = \partial^2_{\hat{y}_i^{(t-1)}}l(y_i, \hat{y}_i^{(t-1)})hi​=∂y^​i(t−1)​2​l(yi​,y^​i(t−1)​)，本质上就是损失的一阶梯度和二阶梯度，由于我们的目标是最小化ttt轮的误差，所以可以去掉与ftf_tft​无关的量

L~(t)=∑i=1n[gift(xi)+12hift2(xi)]+Ω(ft)\tilde{\mathcal{L}}^{(t)} = \sum^n_{i=1}[g_if_t(\boldsymbol{x}_i) + \frac{1}{2}h_if_t^2(\boldsymbol{x}_i)] + \Omega(f_t) L~(t)=i=1∑n​[gi​ft​(xi​)+21​hi​ft2​(xi​)]+Ω(ft​)

定义Ij={i∣q(xi)=j}I_j = \{ i|q(\boldsymbol{x}_i) = j \}Ij​={i∣q(xi​)=j}为划分到index为jjj的叶结点的样本的index集合（简单来说就是统计落入jjj叶结点的所有样本），那么重写上式为

L~(t)=∑i=1n[gift(xi)+12hift2(xi)]+γT+12λ∑j=1Twj2=∑j=1T[(∑i∈Ijgi)wj+12(∑i∈Ijhi+λ)wj2]+γT\tilde{\mathcal{L}}^{(t)} = \sum^n_{i=1}[g_if_t(\boldsymbol{x}_i) + \frac{1}{2}h_if_t^2(\boldsymbol{x}_i)] + \gamma T + \frac{1}{2}\lambda \sum^T_{j=1}w_j^2 \\ = \sum^T_{j=1}[(\sum_{i \in I_j}g_i)w_j + \frac{1}{2}(\sum_{i \in I_j}h_i + \lambda)w_j^2] + \gamma T L~(t)=i=1∑n​[gi​ft​(xi​)+21​hi​ft2​(xi​)]+γT+21​λj=1∑T​wj2​=j=1∑T​[(i∈Ij​∑​gi​)wj​+21​(i∈Ij​∑​hi​+λ)wj2​]+γT

上式的两种变形考虑的对象不同，第一个式子考虑每个样本，第二个式子考虑的是当前树的每个叶结点（由于每个样本必定落入某个叶结点，因此考虑叶结点简化了计算过程），通过对wjw_jwj​求偏导数使损失最小

wj∗=−∑i∈Ijgi∑i∈Ijhi+λw^*_j = - \frac{\sum_{i \in I_j} g_i}{\sum_{i \in I_j}h_i + \lambda} wj∗​=−∑i∈Ij​​hi​+λ∑i∈Ij​​gi​​

一般令Gj=∑i∈Ijgi,Hj=∑i∈IjhiG_j = \sum_{i \in I_j} g_i, H_j = \sum_{i \in I_j}h_iGj​=∑i∈Ij​​gi​,Hj​=∑i∈Ij​​hi​，所以上式为

wj∗=−GjHj+λw^*_j = - \frac{G_j}{H_j + \lambda} wj∗​=−Hj​+λGj​​

将最优wj∗w_j^*wj∗​代入到损失函数中，我们得到与树结构qqq相关的损失函数

L~(t)(q)=−12∑j=1T(∑i∈Ijgi)2∑i∈Ijhi+λ+γT\tilde{\mathcal{L}}^{(t)}(q) = -\frac{1}{2}\sum^T_{j=1}\frac{(\sum_{i \in I_j}g_i)^2}{\sum_{i \in I_j}h_i + \lambda} + \gamma T L~(t)(q)=−21​j=1∑T​∑i∈Ij​​hi​+λ(∑i∈Ij​​gi​)2​+γT

因此上式可以作为衡量当前决策树性能的分数函数，通常用ObjObjObj表示。

如何确定最优树结构，理论上遍历所有树结构可以得到最小损失，但是很明显实际不可行。通常从一个根结点出发，逐渐增加左右两个分支，令ILI_LIL​和IRI_RIR​分别表示落入到左子树和右子树的样本集，计算损失下降Lsplit\mathcal{L}_{split}Lsplit​

Lsplit=12[(∑i∈ILgi)2∑i∈ILhi+λ+(∑i∈IRgi)2∑i∈IRhi+λ−(∑i∈Igi)2∑i∈Ihi+λ]−λ\mathcal{L}_{split} = \frac{1}{2}[\frac{(\sum_{i \in I_L}g_i)^2}{\sum_{i \in I_L}h_i + \lambda} + \frac{(\sum_{i \in I_R}g_i)^2}{\sum_{i \in I_R}h_i + \lambda} - \frac{(\sum_{i \in I}g_i)^2}{\sum_{i \in I}h_i + \lambda}] - \lambda Lsplit​=21​[∑i∈IL​​hi​+λ(∑i∈IL​​gi​)2​+∑i∈IR​​hi​+λ(∑i∈IR​​gi​)2​−∑i∈I​hi​+λ(∑i∈I​gi​)2​]−λ

这个式子很好理解，就是未分支损失-分枝后损失（λ\lambdaλ系数为1是由于仅增加了一个叶结点），所以这个值越大说明分支效果越好。

3. 收缩与特征采样

原文是Shrinkage和Column Subsampling，是两种避免过拟合的策略

  • Shrinkage通过参数η\etaη限制新加入的树的权重www，目的是为了避免新加入的树性能过强导致后面加入的树没有优化空间，从而避免过拟合；
  • Column Subsampling在随机森林中也使用过，先从当前结点的属性集合中随机选择一个包含k个属性的子集，然后再从这个子集中选择最优属性用于划分。

4. 分支划分算法

4.1 精确贪心算法

原文是Exact Greedy Algorithm，这个算法在决策树连续值处理里面介绍过，通过遍历每个特征，寻找各个特征的最佳划分阈值，最后取能使Lsplit\mathcal{L}_{split}Lsplit​分数最大的特征及阈值。

  • 在XGBoost上面的介绍中已经说明采用了Column Subsampling策略，因此在确定某一棵树的结构时并不会遍历整个特征集，而在从特征集中采样出一个子集，这里子集的大小为mmm；
  • 需要将第kkk个特征的所有样本按照大小排序后再计算阈值（如果是连续值，排序后取相邻样本的均值作为待定阈值；如果是离散值，那么每次选择其中一个）。

4.2 近似算法

Exact Greedy Algorithm存在两个问题：一是遍历大数据集是非常不可取的，二是不可能应用于分布式环境中，因此提出Approximate Algorithm用于解决上述两个问题。

相较于Exact Greedy Algorithm遍历所有可能的阈值，在Approximate Algorithm中直接将第kkk个特征的阈值集合SkS_kSk​按百分比分类列出来，形成候选集合，然后类似Exact Greedy Algorithm中找到分数最大值对应的划分方式，按照近似算法得到的结果往往与精确贪心算法相差不大。

4.3 加权分位数草图

原文是Weighted Quantile Sketch，在近似算法中需要选出候选子集，那么如何以一个相对合理的方式划分出候选子集？定义数据集Dk={(x1k,h1),(x2k,h2),...,(xnk,hn)}\mathcal{D}_k = \{ (x_{1k}, h_1), (x_{2k}, h_2),..., (x_{nk}, h_n) \}Dk​={(x1k​,h1​),(x2k​,h2​),...,(xnk​,hn​)}包括第kkk个特征的值xkx_kxk​以及二阶梯度hhh，且按照顺序排列好，则定义排序函数rk:R→[0,+∞)r_k: \mathbb{R} \rightarrow [0, +\infty)rk​:R→[0,+∞)

rk(z)=1∑(x,h)∈Dkh∑(x,h)∈Dk,x<zhr_k(z) = \frac{1}{\sum_{(x, h) \in \mathcal{D}_k}h} \sum_{(x, h) \in \mathcal{D}_k, x<z} h rk​(z)=∑(x,h)∈Dk​​h1​(x,h)∈Dk​,x<z∑​h

则最佳划分候选子集{sk1,sk2,...,skl}\{s_{k1}, s_{k2}, ..., s_{kl}\}{sk1​,sk2​,...,skl​}为

∣rk(sk,j)−rk(sk,j+1)∣<ϵ,sk1=min⁡ixik,skl=max⁡ixik|r_k(s_{k,j}) - r_k(s_{k,j+1})|<\epsilon, \quad s_{k1} = \underset{i}{\min} x_{ik}, s_{kl} = \underset{i}{\max} x_{ik} ∣rk​(sk,j​)−rk​(sk,j+1​)∣<ϵ,sk1​=imin​xik​,skl​=imax​xik​

对上式的理解为，若排序后的样本的二阶梯度占所有样本的二阶梯度和的比例小于ϵ\epsilonϵ，则认为这个区间内的样本属于同一区域，那么我们取这个区域的最大值和最小值作为候选子集的首尾（同样取均值作为划分阈值）。

ϵ\epsilonϵ是近似因子，一般意味候选子集大小为1ϵ\frac{1}{\epsilon}ϵ1​，也就是说通过控制ϵ\epsilonϵ的大小我们可以控制候选子集的大小，同时也是划分的精度，从这个角度来看优化目标，我们需要二阶梯度尽可能小的样本作为候选子集，由wj∗w^*_jwj∗​计算公式表明样本xi\boldsymbol{x}_ixi​的最优权重可以表示为gihi\frac{g_i}{h_i}hi​gi​​。我们可以重写第ttt棵树的优化目标为

∑i=1n12hi(ft(xi)−gihi)2+Ω(ft)+constant\sum^n_{i=1}\frac{1}{2}h_i(f_t(\boldsymbol{x}_i) - \frac{g_i}{h_i})^2 + \Omega(f_t) + constant i=1∑n​21​hi​(ft​(xi​)−hi​gi​​)2+Ω(ft​)+constant

重写后的优化目标将hih_ihi​视为样本权重，gihi\frac{g_i}{h_i}hi​gi​​视为真实标签。

4.4 稀疏划分

原文是Sparsity-aware Split Finding，在现实任务中会遇到稀疏矩阵的问题，有可能是数据缺失，有可能是数据属于onehot变量，我们需要能处理稀疏数据的方式，一种简单的方式是制定一个默认的方向，当遇到数据缺失时使其具有默认的方向。一般有两种方式设定数据的方向：

  • 从数据中学到划分方向，这里参照下图，在计算左子树分支损失时将缺失值的样本作为右子树的集合，在计算右子树分支损失时将缺失值的样本作为左子树的集合；
  • 人为指定方向（或者不采用带有缺失值的数据）。
