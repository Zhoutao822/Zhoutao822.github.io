《深度学习》第13章 线性因子模型

相较于线性模型描述输入x\boldsymbol{x}x与输出y\boldsymbol{y}y的关系，线性因子模型考虑的是输入x\boldsymbol{x}x与隐变量h\boldsymbol{h}h的关系，也就是说线性因子模型的目的是发掘出输入变量x\boldsymbol{x}x的统计特征，利用隐变量实现降维或输入优化等目的。

线性因子模型，顾名思义，我们假设x\boldsymbol{x}x是h\boldsymbol{h}h的线性变换以及添加噪声来生成的

x=Wh+b+noise\boldsymbol{x} = \boldsymbol{Wh} + \boldsymbol{b} + noise x=Wh+b+noise

其中解释性因子h\boldsymbol{h}h从一个分布中抽取

h∼p(h)\boldsymbol{h} \sim p(\boldsymbol{h}) h∼p(h)

其中p(h)p(\boldsymbol{h})p(h)是一个因子分布，满足p(h)=∏ip(hi)p(\boldsymbol{h}) = \prod_ip(h_i)p(h)=∏i​p(hi​)，噪声通常是对角化的且服从高斯分布。

1. 概率PCA和因子分析

概率PCA、因子分析和其他线性因子模型是上述等式的特殊情况，并且仅在对观测到x\boldsymbol{x}x之前的噪声分布和潜变量h\boldsymbol{h}h先验的选择上有所不同。

1.1 因子分析

在因子分析中，潜变量的先验是一个方差为单位矩阵的高斯分布

h∼N(h;0,I)\boldsymbol{h} \sim N(\boldsymbol{h};\boldsymbol{0},\boldsymbol{I}) h∼N(h;0,I)

同时假定在给定h\boldsymbol{h}h的条件下观察值xix_ixi​是条件独立的。具体来说，我们可以假设噪声是从对角协方差矩阵的高斯分布中抽出的，协方差矩阵为ψ=diag(σ2)\boldsymbol{\psi} = diag(\boldsymbol{\sigma}^2)ψ=diag(σ2)，其中σ2=[σ12,σ22,...,σn2]T\boldsymbol{\sigma}^2 = [\sigma_1^2, \sigma_2^2, ...,\sigma_n^2]^Tσ2=[σ12​,σ22​,...,σn2​]T表示一个向量，每个元素表示一个变量的方差。

因此潜变量的作用是捕获不同观测变量xix_ixi​之间的依赖关系。

x∼N(x;b,WWT+ψ)\boldsymbol{x} \sim N(\boldsymbol{x};\boldsymbol{b}, \boldsymbol{W}\boldsymbol{W}^T + \boldsymbol{\psi}) x∼N(x;b,WWT+ψ)

1.2 概率PCA

对因子分析模型作轻微修改，使条件方差σi2\sigma_i^2σi2​等于同一个值。在这种情况下，x\boldsymbol{x}x的协方差简化为WWT+σ2I)\boldsymbol{W}\boldsymbol{W}^T + \sigma^2\boldsymbol{I})WWT+σ2I)，这里的σ2\sigma^2σ2是一个标量。

x∼N(x;b,WWT+σ2I)\boldsymbol{x} \sim N(\boldsymbol{x};\boldsymbol{b}, \boldsymbol{W}\boldsymbol{W}^T + \sigma^2\boldsymbol{I}) x∼N(x;b,WWT+σ2I)

或等价

x=Wh+b+σz\boldsymbol{x} = \boldsymbol{Wh} + \boldsymbol{b} + \sigma \boldsymbol{z} x=Wh+b+σz

其中z∼N(z;0,I)\boldsymbol{z} \sim N(\boldsymbol{z}; \boldsymbol{0}, \boldsymbol{I})z∼N(z;0,I)是高斯噪声，Tipping and Bishop提出了一种迭代的EM算法来估计参数W\boldsymbol{W}W和σ2\sigma^2σ2。

概率PCA模型利用了一种观察现象：除了一些微小残余的重构误差（至多为σ2\sigma^2σ2），数据中的大多数变量可以由潜变量h\boldsymbol{h}h描述。当σ→0\sigma \rightarrow 0σ→0，概率PCA退化为PCA，概率PCA所定义的密度函数在ddd维的W\boldsymbol{W}W的列空间周围非常尖锐。这导致模型会为没有在一个超平面附近聚集的数据分配非常低的概率。

2. 独立成分分析

独立成分分析ICA是一种建模线性因子的方法，旨在将观察到的信号分离成许多潜在信号，这些潜在信号通过压缩和叠放可以恢复成观察数据。这些信号是完全独立，而不是仅仅彼此不相关。

潜在因子h\boldsymbol{h}h的先验p(h)p(\boldsymbol{h})p(h)，必须由用户提前给出并固定。接着模型确定性地生成x=Wh\boldsymbol{x} = \boldsymbol{Wh}x=Wh。我们可以通过非线性变化来确定p(x)p(\boldsymbol{x})p(x)。然后通过一般的方法比如最大化似然进行学习。

ICA的所有变种均要求p(h)p(\boldsymbol{h})p(h)是非高斯的。因为若p(h)p(\boldsymbol{h})p(h)具有高斯分量的独立先验，对于许多W\boldsymbol{W}W值，我们可以在p(x)p(\boldsymbol{x})p(x)上获得相同的分布，那么W\boldsymbol{W}W存在多解，意味着不可解。在用户明确指定分布的最大似然方法中，一个典型的选择是使用p(hi)=∂∂hiσ(hi)p(h_i) = \frac{\partial}{\partial h_i}\sigma(h_i)p(hi​)=∂hi​∂​σ(hi​)。这些非高斯分布的典型选择在0附近具有比高斯分布更高的峰值，因此我们也可以看到独立成分分析经常用于学习稀疏特征。

3. 慢特征分析

慢特征分析SFA是使用来自时间信号的信息学习不变特征的线性因子模型。

慢特征分析的想法源于所谓的慢性原则（slowness principle）。其基本思想是，与场景中起描述作用的单个量度相比，场景的重要特性通常变化得非常缓慢。例如，在计算机视觉中，单个像素值可以非常快速地改变。如果斑马从左到右移动穿过图像并且它的条纹穿过对应的像素时，该像素将迅速从黑色变为白色，并再次恢复成黑色。通过比较，指示斑马是否在图像中的特征将不发生改变，并且描述斑马位置的特征将缓慢地改变。因此，我们可能希望将模型正则化，从而能够学习到那些随时间变化较为缓慢的特征。

一般来说，我们可以将慢性原则应用于可以使用梯度下降训练的任何可微分模型。为了引入慢性原则，我们可以向代价函数添加以下项

λ∑tL(f(x(t+1)),f(x(t)))\lambda \sum_t L(f(\boldsymbol{x}^{(t+1)}), f(\boldsymbol{x}^{(t)})) λt∑​L(f(x(t+1)),f(x(t)))

其中λ\lambdaλ是确定慢度正则化强度的超参数项，ttt是样本时间序列的索引，fff是需要正则化的特征提取器，LLL是测量f(x(t))f(x(t))f(x(t))和f(x(t+1))f(x(t+1))f(x(t+1))之间的距离的损失函数。LLL的一个常见选择是均方误差。

SFA算法先将f(x;θ)f(\boldsymbol{x};\theta)f(x;θ)定义为线性变换，然后求解如下优化问题

min⁡θEt(f(x(t+1))i−f(x(t))i)2\underset{\boldsymbol{\theta}}{\min} \mathbb{E}_t(f(\boldsymbol{x}^{(t+1)})_i - f(\boldsymbol{x}^{(t)})_i)^2 θmin​Et​(f(x(t+1))i​−f(x(t))i​)2

并满足下面的约束

Etf(x(t))i=0Et[f(x(t))i2]=1\mathbb{E}_tf(\boldsymbol{x}^{(t)})_i = 0 \\ \mathbb{E}_t[f(\boldsymbol{x}^{(t)})_i^2] = 1 Et​f(x(t))i​=0Et​[f(x(t))i2​]=1

学习特征具有零均值的约束对于使问题具有唯一解是必要的；否则我们可以向所有特征值添加一个常数，并获得具有相等慢度目标值的不同解。特征具有单位方差的约束对于防止所有特征趋近于0的病态解是必要的。

与 PCA 类似，SFA 特征是有序的，其中学习第一特征是最慢的。要学习多个特征，我们还必须添加约束

∀i<j,Et[f(x(t))if(x(t))j]=0\forall i < j, \quad \mathbb{E}_t[f(\boldsymbol{x}^{(t)})_if(\boldsymbol{x}^{(t)})_j] = 0 ∀i<j,Et​[f(x(t))i​f(x(t))j​]=0

这要求学习的特征必须彼此线性去相关。没有这个约束，所有学习到的特征将简单地捕获一个最慢的信号。可以想象使用其他机制，如最小化重构误差，也可以迫使特征多样化。但是由于 SFA 特征的线性，这种去相关机制只能得到一种简单的解。SFA 问题可以通过线性代数软件获得闭式解。

深度 SFA 也已经被用于学习用在对象识别和姿态估计的特征。

4. 稀疏编码

稀疏编码是一个线性因子模型，像大多数其他线性因子模型一样，它使用了线性的解码器加上噪声的方式获得一个x\boldsymbol{x}x的重构，稀疏编码模型通常假设线性因子有一个各向同性精度为β\betaβ的高斯噪声

p(x∣h)=N(x;Wh+b,1βI)p(\boldsymbol{x}|\boldsymbol{h}) = N(\boldsymbol{x};\boldsymbol{Wh} + \boldsymbol{b}, \frac{1}{\beta}\boldsymbol{I}) p(x∣h)=N(x;Wh+b,β1​I)

分布p(h)p(\boldsymbol{h})p(h)通常选取为一个峰值很尖锐且接近0的分布。常见的选择包括可分解的Laplace、Cauchy或者可分解的Student-t分布。例如，以稀疏惩罚系数λ\lambdaλ为参数的Laplace先验可以表示为

p(hi)=Laplace(hi;0,2λ)=λ4e−12λ∣hi∣p(h_i) = Laplace(h_i;0,\frac{2}{\lambda}) = \frac{\lambda}{4}e^{-\frac{1}{2}\lambda|h_i|} p(hi​)=Laplace(hi​;0,λ2​)=4λ​e−21​λ∣hi​∣

相应的，Student-t先验分布可以表示为

p(hi)∝1(1+hi2v)v+12p(h_i) \propto \frac{1}{(1 + \frac{h_i^2}{v})^{\frac{v+1}{2}}} p(hi​)∝(1+vhi2​​)2v+1​1​

使用最大似然的方法来训练稀疏编码模型是不可行的。相反，为了在给定编码的情况下更好地重构数据，训练过程在编码数据和训练解码器之间交替进行。

编码器是一个优化算法，在这个优化问题中，我们寻找单个最可能的编码值：

h∗=f(x)=arg⁡max⁡hp(h∣x)\boldsymbol{h}^* = f(\boldsymbol{x}) = \underset{\boldsymbol{h}}{\arg \max} p(\boldsymbol{h}|\boldsymbol{x}) h∗=f(x)=hargmax​p(h∣x)

优化问题如下

arg⁡max⁡hp(h∣x)=arg⁡max⁡hlog⁡p(h∣x)=arg⁡min⁡hλ∣∣h∣∣1+β∣∣x−Wh∣∣22\underset{\boldsymbol{h}}{\arg \max} p(\boldsymbol{h}|\boldsymbol{x}) \\ = \underset{\boldsymbol{h}}{\arg \max} \log p(\boldsymbol{h}|\boldsymbol{x}) \\ = \underset{\boldsymbol{h}}{\arg \min} \lambda ||\boldsymbol{h}||_1 + \beta ||\boldsymbol{x} - \boldsymbol{Wh}||^2_2 hargmax​p(h∣x)=hargmax​logp(h∣x)=hargmin​λ∣∣h∣∣1​+β∣∣x−Wh∣∣22​

由于L1L_1L1​范数，这个过程将产生稀疏的h∗\boldsymbol{h}^*h∗。

为了训练模型而不仅仅是进行推断，我们交替迭代关于h\boldsymbol{h}h和W\boldsymbol{W}W的最小化过程。在本文中，我们将β\betaβ视为超参数。我们通常将其设置为 1，因为它在此优化问题的作用与λ\lambdaλ类似，没有必要使用两个超参数。

5. PCA的流形解释

线性因子模型，包括 PCA 和因子分析，可以理解为学习一个流形。

我们可以将概率 PCA 定义为高概率的薄饼状区域，即一个高斯分布，沿着某些轴非常窄，就像薄饼沿着其垂直轴非常平坦，但沿着其他轴是细长的，正如薄饼在其水平轴方向是很宽的一样。

PCA 可以理解为将该薄饼与更高维空间中的线性流形对准。这种解释不仅适用于传统 PCA，而且适用于学习矩阵W\boldsymbol{W}W和V\boldsymbol{V}V的任何线性自编码器，其目的是使重构的x\boldsymbol{x}x尽可能接近于原始的x\boldsymbol{x}x。

编码器表示为

h=f(x)=WT(x−μ)\boldsymbol{h} = f(\boldsymbol{x}) = \boldsymbol{W}^T(\boldsymbol{x} - \boldsymbol{\mu}) h=f(x)=WT(x−μ)

编码器计算h\boldsymbol{h}h的低维表示，解码器负责重构

x^=g(h)=b+Vh\hat{\boldsymbol{x}} = g(\boldsymbol{h}) = \boldsymbol{b} + \boldsymbol{Vh} x^=g(h)=b+Vh

能够最小化重构误差

E[∣∣x−x^∣∣2]\mathbb{E}[||\boldsymbol{x} - \hat{\boldsymbol{x}}||^2] E[∣∣x−x^∣∣2]

的线性编码器和解码器的选择对应着V=W,μ=b=E[x]\boldsymbol{V} = \boldsymbol{W}, \boldsymbol{\mu} = \boldsymbol{b} = \mathbb{E}[\boldsymbol{x}]V=W,μ=b=E[x]，W\boldsymbol{W}W的列形成一组标准正交基，这组基生成的子空间与协方差矩阵C\boldsymbol{C}C

C=E[(x−μ)(x−μ)T]\boldsymbol{C} = \mathbb{E}[(\boldsymbol{x} - \boldsymbol{\mu})(\boldsymbol{x} - \boldsymbol{\mu})^T] C=E[(x−μ)(x−μ)T]

的主特征向量所生成的子空间相同。在PCA中，W\boldsymbol{W}W的列是按照对应特征值幅度大小排序所对应的特征向量。

我们还可以发现C\boldsymbol{C}C的特征值λi\lambda_iλi​对应了x\boldsymbol{x}x在特征向量v(i)\boldsymbol{v}^{(i)}v(i)方向上的方差。如果x∈RD,h∈Rd\boldsymbol{x} \in \mathbb{R}^D, \boldsymbol{h} \in \mathbb{R}^dx∈RD,h∈Rd并且满足d<Dd < Dd<D，则最佳的重构误差是

min⁡E[∣∣x−x^∣∣2]=∑i=d+1Dλi\min \mathbb{E}[||\boldsymbol{x} - \hat{\boldsymbol{x}}||^2] = \sum^D_{i = d+1} \lambda_i minE[∣∣x−x^∣∣2]=i=d+1∑D​λi​

因此，如果协方差矩阵的秩为ddd，则特征值λd+1\lambda_{d+1}λd+1​到λD\lambda_DλD​都为0，并且重构误差为0。

此外，我们还可以证明上述解可以通过在给定正交矩阵W\boldsymbol{W}W的情况下最大化h\boldsymbol{h}h元素的方差而不是最小化重构误差来获得。
