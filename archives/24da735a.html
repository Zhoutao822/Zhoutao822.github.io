1. 高斯分布

高斯分布又叫正态分布，在现实生活中充满了高斯分布，比如考试的分数、人的身高是一维高斯分布，图像处理领域的高斯模糊是二维高斯分布等等，通过高斯分布，我们可以很好的描述一类随机变量。

对于连续型标量随机变量xxx来说，若xxx服从正态分布，则其概率密度函数是

p(x)=1(2π)1/2σexp⁡[−12(x−μσ)2]=N(x;μ,σ2)(−∞<x<∞;σ>0)p(x) = \frac{1}{(2\pi)^{1/2}\sigma} \exp[-\frac{1}{2}(\frac{x - \mu}{\sigma})^2] = N(x; \mu, \sigma^2) \\ (-\infty < x < \infty; \sigma > 0) p(x)=(2π)1/2σ1​exp[−21​(σx−μ​)2]=N(x;μ,σ2)(−∞<x<∞;σ>0)

对于多元正态随机变量x=(x1,x2,...,xD)T\boldsymbol{x} = (x_1, x_2,...,x_D)^Tx=(x1​,x2​,...,xD​)T，其联合概率密度

p(x)=1(2π)D/2∣Σ∣1/2exp⁡[−12(x−μ)TΣ−1(x−μ)]=N(x;μ,Σ)其中μ=E(x)∈RD;Σ=E[(x−xˉ)(x−xˉ)T]∈RD×Dp(\boldsymbol{x}) = \frac{1}{(2\pi)^{D/2}|\boldsymbol{\Sigma}|^{1/2}} \exp[-\frac{1}{2}(\boldsymbol{x} - \boldsymbol{\mu})^T\boldsymbol{\Sigma}^{-1}(\boldsymbol{x}-\boldsymbol{\mu})] = N(\boldsymbol{x};\boldsymbol{\mu}, \boldsymbol{\Sigma}) \\ 其中 \boldsymbol{\mu} = E(\boldsymbol{x}) \in \mathbb{R}^D; \boldsymbol{\Sigma} = E[(\boldsymbol{x} - \bar{\boldsymbol{x}})(\boldsymbol{x} - \bar{\boldsymbol{x}})^T] \in \mathbb{R}^{D \times D} p(x)=(2π)D/2∣Σ∣1/21​exp[−21​(x−μ)TΣ−1(x−μ)]=N(x;μ,Σ)其中μ=E(x)∈RD;Σ=E[(x−xˉ)(x−xˉ)T]∈RD×D

对于一个标量连续随机变量x，若其服从混合高斯分布，其概率密度函数

p(x)=∑m=1Mcm(2π)1/2σmexp⁡[−12(x−μmσm)2]=∑m=1McmN(x;μm,σm2)，其中∑m=1Mcm=1，E(x)=∑m=1Mcmμmp(x) = \sum^{M}_{m=1}\frac{c_m}{(2\pi)^{1/2}\sigma_m}\exp[-\frac{1}{2}(\frac{x-\mu_m}{\sigma_m})^2] \\ = \sum^{M}_{m=1}c_mN(x;\mu_m,\sigma^2_m)，其中\sum^M_{m=1}c_m = 1，E(x) = \sum^{M}_{m=1}c_m\mu_m p(x)=m=1∑M​(2π)1/2σm​cm​​exp[−21​(σm​x−μm​​)2]=m=1∑M​cm​N(x;μm​,σm2​)，其中m=1∑M​cm​=1，E(x)=m=1∑M​cm​μm​

推广到多变量的多元混合高斯分布，其联合概率密度函数为

p(x)=∑m=1Mcm(2π)D/2∣Σm∣1/2exp⁡[−12(x−μm)TΣm−1(x−μm)]=∑m=1McmN(x;μm,Σm)p(\boldsymbol{x}) = \sum^{M}_{m=1}\frac{c_m}{(2\pi)^{D/2}|\boldsymbol{\Sigma}_m|^{1/2}}\exp[-\frac{1}{2}(\boldsymbol{x} - \boldsymbol{\mu}_m)^T\boldsymbol{\Sigma}^{-1}_m(\boldsymbol{x} - \boldsymbol{\mu}_m)] \\ = \sum^{M}_{m=1}c_mN(\boldsymbol{x};\boldsymbol{\mu}_m,\boldsymbol{\Sigma}_m) p(x)=m=1∑M​(2π)D/2∣Σm​∣1/2cm​​exp[−21​(x−μm​)TΣm−1​(x−μm​)]=m=1∑M​cm​N(x;μm​,Σm​)

在多元混合高斯分布中，如果变量x的维度D很大（比如40），那么使用全协方差矩阵（非对角）（Σm\Sigma_mΣm​）将引入大量参数（大约为M×D2M \times D^2M×D2）。为了减少这个数量，可以使用对角协方差矩阵。当M很大时，也可以限制所有的协方差为相同矩阵，对所有的混合成分m，将参数Σm\Sigma_mΣm​绑定在一起。

2. GMM示例

对于这样的二维数据分布，如何进行分类，若使用单个高斯成分，那么一个椭圆可以描述整体，但是根据观察，实际上这些数据应该属于两个不同的高斯成分，也就是说可以分为两类。

那么我们使用两个二维高斯分布来描述这个分布，当然最后的结果使用了cmc_mcm​进行叠加

对于上例，我们定义x=(x1,x2)T，M=2\boldsymbol{x} = (x_1, x_2)^T，M = 2x=(x1​,x2​)T，M=2，则其GMM形式为

p(x)=c1N(μ1,σ1)+c2N(μ2,σ2)p(\boldsymbol{x}) = c_1N(\boldsymbol{\mu}_1, \boldsymbol{\sigma}_1) + c_2N(\boldsymbol{\mu}_2, \boldsymbol{\sigma}_2) p(x)=c1​N(μ1​,σ1​)+c2​N(μ2​,σ2​)

那么我们需要求解的参数为(c1,μ1,σ1;c2,μ2,σ2)(c_1, \boldsymbol{\mu}_1, \boldsymbol{\sigma}_1; c_2, \boldsymbol{\mu}_2, \boldsymbol{\sigma}_2)(c1​,μ1​,σ1​;c2​,μ2​,σ2​)

3. 参数估计

这里使用EM算法（EM算法必收敛）估计参数变量Θ=cm,μm,σm\Theta = {c_m, \boldsymbol{\mu}_m, \boldsymbol{\sigma}_m}Θ=cm​,μm​,σm​，首先回顾一下，GMM表示形式

p(x)=∑m=1McmN(x;μm,σm2)p(\boldsymbol{x}) = \sum^{M}_{m=1}c_mN(\boldsymbol{x};\boldsymbol{\mu}_m,\boldsymbol{\sigma}^2_m) p(x)=m=1∑M​cm​N(x;μm​,σm2​)

其中cmc_mcm​可以看作第m个高斯成分被选中的概率。我们引入一个新的M维变量z\boldsymbol{z}z，zmz_mzm​只能取0或1；zm=1z_m = 1zm​=1表示第m类被选中的概率cmc_mcm​，即p(zm=1)=cm，且∑m=1Mzm=1p(z_m = 1) = c_m，且\sum^M_{m=1} z_m = 1p(zm​=1)=cm​，且∑m=1M​zm​=1，所以z\boldsymbol{z}z的联合概率分布：

p(z)=p(z1)p(z2)...p(zM)=∏m=1Mcmzmp(\boldsymbol{z}) = p(z_1)p(z_2)...p(z_M) = \prod_{m=1}^{M}c_m^{z_m} p(z)=p(z1​)p(z2​)...p(zM​)=m=1∏M​cmzm​​

又因为对于属于某一个高斯成分的数据来说（zm=1z_m = 1zm​=1），其分布必定是高斯分布，所以

p(x∣zm=1)=N(x∣μm,σm)p(\boldsymbol{x}|z_m = 1) = N(\boldsymbol{x}|\boldsymbol{\mu}_m, \boldsymbol{\sigma}_m) p(x∣zm​=1)=N(x∣μm​,σm​)

进而上式等价于

p(x∣z)=∏m=1MN(x∣μm,σm)zmp(\boldsymbol{x}|\boldsymbol{z}) = \prod_{m=1}^{M}N(\boldsymbol{x}|\boldsymbol{\mu}_m, \boldsymbol{\sigma}_m)^{z_m} p(x∣z)=m=1∏M​N(x∣μm​,σm​)zm​

我们有了p(z)和p(x∣z)p(\boldsymbol{z})和p(\boldsymbol{x}|\boldsymbol{z})p(z)和p(x∣z)，那么可以计算p(x)p(\boldsymbol{x})p(x)

p(x)=∑Zp(x∣z)p(z)=∑m=1McmN(x;μm,σm2)，zm=0的项在连乘中为1p(\boldsymbol{x}) = \sum_Z p(\boldsymbol{x}|\boldsymbol{z})p(\boldsymbol{z}) \\ = \sum^{M}_{m=1}c_mN(\boldsymbol{x};\boldsymbol{\mu}_m,\boldsymbol{\sigma}^2_m)，z_m = 0的项在连乘中为1 p(x)=Z∑​p(x∣z)p(z)=m=1∑M​cm​N(x;μm​,σm2​)，zm​=0的项在连乘中为1

所以最终又回到了GMM的表示形式，但是我们增加了隐含变量z用于描述我们已经知道有多少个高斯成分。

在贝叶斯的思想下，p(z)p(\boldsymbol{z})p(z)是先验概率，p(x∣z)p(\boldsymbol{x}|\boldsymbol{z})p(x∣z)是似然概率，我们可以求解出后验概率p(z∣x)p(\boldsymbol{z}|\boldsymbol{x})p(z∣x)。

E步：基于当前迭代轮数（用j表示），针对某个高斯成分m，用给定观察值x(t)\boldsymbol{x}^{(t)}x(t)计算得到的后验概率t=1,...,N，（N是采样率）t = 1, ..., N，（N是采样率）t=1,...,N，（N是采样率）

hm(j)(t)=p(zm(j)=1∣x(t))=p(zm(j)=1)p(x(t)∣zm(j)=1)p(x(t))=p(zm(j)=1)p(x(t)∣zm(j)=1)∑i=1Mp(zi=1)p(x(t)∣zi=1)=cm(j)N(x(t);μm(j),σm(j))∑i=1Mci(j)N(x(t);μi(j),σi(j))h^{(j)}_m(t) = p(z_m^{(j)}=1|\boldsymbol{x}^{(t)}) \\ = \frac{p(z_m^{(j)}=1)p(\boldsymbol{x}^{(t)}|z_m^{(j)}=1)}{p(\boldsymbol{x}^{(t)})} \\ = \frac{p(z_m^{(j)}=1)p(\boldsymbol{x}^{(t)}|z_m^{(j)}=1)}{\sum^M_{i=1}p(z_i=1)p(\boldsymbol{x}^{(t)}|z_i=1)} \\ = \frac{c_m^{(j)}N(\boldsymbol{x}^{(t)}; \boldsymbol{\mu}_m^{(j)}, \boldsymbol{\sigma}_m^{(j)})}{\sum^M_{i=1}c_i^{(j)}N(\boldsymbol{x}^{(t)}; \boldsymbol{\mu}_i^{(j)}, \boldsymbol{\sigma}_i^{(j)})} hm(j)​(t)=p(zm(j)​=1∣x(t))=p(x(t))p(zm(j)​=1)p(x(t)∣zm(j)​=1)​=∑i=1M​p(zi​=1)p(x(t)∣zi​=1)p(zm(j)​=1)p(x(t)∣zm(j)​=1)​=∑i=1M​ci(j)​N(x(t);μi(j)​,σi(j)​)cm(j)​N(x(t);μm(j)​,σm(j)​)​

cm(j+1)=1N∑t=1Nhm(j)(t)μm(j+1)=∑t=1Nhm(j)(t)x(t)∑t=1Nhm(j)(t)σm(j+1)=∑t=1Nhm(j)(t)[x(t)−μm(j)][x(t)−μm(j)]T∑t=1Nhm(j)(t)c_m^{(j + 1)} = \frac{1}{N}\sum^N_{t=1}h_m^{(j)}(t) \\ \boldsymbol{\mu}_m^{(j + 1)} = \frac{\sum^N_{t=1}h_m^{(j)}(t)\boldsymbol{x}^{(t)}}{\sum^N_{t=1}h_m^{(j)}(t)} \\ \boldsymbol{\sigma}_m^{(j + 1)} = \frac{\sum^N_{t=1}h_m^{(j)}(t)[\boldsymbol{x}^{(t)} - \boldsymbol{\mu}_m^{(j)}][\boldsymbol{x}^{(t)} - \boldsymbol{\mu}_m^{(j)}]^T}{\sum^N_{t=1}h_m^{(j)}(t)} cm(j+1)​=N1​t=1∑N​hm(j)​(t)μm(j+1)​=∑t=1N​hm(j)​(t)∑t=1N​hm(j)​(t)x(t)​σm(j+1)​=∑t=1N​hm(j)​(t)∑t=1N​hm(j)​(t)[x(t)−μm(j)​][x(t)−μm(j)​]T​

计算对数似然函数

ln⁡p(x;c,μ,σ)=∑t=1Nln⁡(∑m=1McmN(x(t)∣μm,σm))\ln p(\boldsymbol{x};c, \boldsymbol{\mu}, \boldsymbol{\sigma}) = \sum^N_{t=1}\ln{(\sum^M_{m=1}c_mN(\boldsymbol{x}^{(t)}|\boldsymbol{\mu}_m, \boldsymbol{\sigma}_m))} lnp(x;c,μ,σ)=t=1∑N​ln(m=1∑M​cm​N(x(t)∣μm​,σm​))
